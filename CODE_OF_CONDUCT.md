# 行動規範 / Code of Conduct

## 私たちの約束

私たちはオープンで歓迎される環境を作るために、貢献者およびメンテナ全員が、
年齢・体格・障がい・民族性・性自認と表現・経験の程度・国籍・個人的な見た目・
人種・宗教・性的指向に関わらず、ハラスメントの無いコミュニティへの参加を約束します。

## 私たちの基準

ポジティブな行動の例:

- 歓迎的で包括的な言葉遣いをする
- 異なる視点や経験を尊重する
- 建設的な批判を素直に受け入れる
- コミュニティにとって最善のことに焦点を合わせる
- 他のメンバーに共感を示す

受け入れられない行動の例:

- 性的な表現や画像の使用、望まない性的な注目・誘いかけ
- トローリング、侮辱・軽蔑的なコメント、個人的・政治的な攻撃
- 公開・非公開を問わないハラスメント
- 明示的な許可なく他者の個人情報（住所・電子メールアドレス等）を公開する
- その他、職業的な場として不適切と考えられる行為

## 責任

プロジェクトのメンテナは、受け入れられる行動基準を明確化する責任を持ち、
受け入れられない行動に対して適切かつ公平な是正措置を取ることが期待されます。

## 適用範囲

この行動規範は、プロジェクト空間および個人がプロジェクトまたはコミュニティを
代表しているときの公共空間の両方に適用されます。

## 報告

不適切な行動は、プロジェクトの `SECURITY.md` に記載された連絡先、または
各プロジェクトで別途指定されている窓口へ連絡してください。すべての苦情は
レビュー・調査され、状況に応じて必要と判断される対応が取られます。
